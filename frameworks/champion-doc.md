# Champion Document Template

The 1-page document the champion walks into the budget meeting. Account-specific, data-driven, internally shareable. They forward it — they don't rewrite it.

## Principles

1. **Account-specific.** Every number is from their data.
2. **Neutral branding.** Reads like an internal memo, not a vendor pitch.
3. **Numbers first.** Data carries the argument.
4. **One page.** If it doesn't fit, cut.
5. **Forwardable.** No jargon, no context needed beyond what's on the page.

## Template

[Company] [Domain] Assessment — [Date]

HEADLINE: [Trigger event or deadline]. [Company] has [X] [findings] across [Y] [scope units].

[KEY METRIC 1]: [Value] [unit] — [one-sentence impact]
[KEY METRIC 2]: [Value] [unit] — [one-sentence impact]
[KEY METRIC 3]: [Value] [unit] — [one-sentence impact]

COST OF STATUS QUO: [Current approach] covers [X]% of [scope] per [period] at [cost]. [Consequence of gap].

RECOMMENDED PATH: [Solution] covering 100% of [scope] in [timeframe]. Initial engagement: [price]. Ongoing: [subscription]/[period].

EXPECTED OUTCOME: [Key metric] reduced from [current] to [target] within [timeframe]. Full coverage maintained automatically.

TOP FINDINGS:
[Most impactful — specific, with evidence]
[Second — specific]
[Third — specific]

NEXT STEP: [Specific action — named person, proposed date]

## Quality Checks

- [ ] Every number from actual data (no placeholders)
- [ ] No banned language from document quality rules
- [ ] Uses their terminology, not yours
- [ ] Fits one page
- [ ] Specific next step with name and date
- [ ] A non-specialist could understand it
