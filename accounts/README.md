# Accounts

One markdown file per key account. Populate as you research. Each file accumulates findings, stakeholders, pain points, competitive context, objection history, call debriefs.

Naming: lowercase company name, e.g. `acme-corp.md`, `contoso-industries.md`.

Starting template:

```markdown
# [Account Name]

**Last updated:** YYYY-MM-DD

## Overview
- Industry:
- HQ:
- AE/CSM:
- Current ARR:
- Key contacts:

## Current State
- Active deals:
- Renewal dates:
- Usage health:

## Layer 1 (public knowledge)
[from annual reports, website, press]

## Layer 2 (industry context)
[regulatory trends, peer behavior, competitive landscape]

## Layer 3 (first-principles insight)
[your original observations — this is where value compounds]

## Active Threads
[ongoing email threads, open questions, scheduled meetings]

## Call History
[7-point debriefs per frameworks/call-debrief.md]

## Objections Heard
[log per frameworks/objections.md]
```
